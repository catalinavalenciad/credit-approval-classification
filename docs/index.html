Credit Approval Classification: Model Validation and Generalization Analysis

This project demonstrates the design, validation, and comparison of binary classification models for an automated credit approval decision system. The focus is on methodological rigor proper validation design, and unbiased generalization performance rather than in-sample accuracy.

Business Context

Credit approval is a high-stakes decision problem. While this project does not encode explicit cost asymmetry, it emphasizes validation strategies required for trustworthy deployment in regulated decision systems.

Dataset

  • UCI Credit Approval Dataset
  • Dataset
  • 654 observations
  • 10 numeric predictors
  • Binary response variable

Methods

  • Artificial missing-data imputation study
  • Baseline classifiers: Linear SVM and KNN
  • Cross-validation and train/validation/test split
  • Final model selection on a held-out test set

Final Results

  • SVM (Linear): Test accuracy = 0.8626
  • KNN (k = 10): Test accuracy = 0.8779
  • Selected model: KNN
